HTML atau Hypertext Markup Languange adalah bahasa markah yang digunakan untuk membuat sebuah halaman web dan memberitahu browser bagaimana menampiklan konten berupa teks, gambar, tabel form, hyperlink, dll

Css adalah bahasa stylesheet yang digunkan untuk mengatur tampilan dokumen. ada dua sifat CSS yaitu internal dan eksternal. CSS internal dimasukkan secara langsung kedokumen web yang akan didesain.
